## Summary

Portfolio application developed in Next.js and Tailwind CSS and hosted in vercel.

## Features

- Responsive
- Jumbotron buttons for contact
- Reusable skill tiles with ratings
- Portfolio projects with demo and code buttons
- Smooth scrolling
